## üìñ Overview

This project is the frontend repository for the **lom 10th Anniversary** celebration website. It is designed to showcase the history, events, and memories of the past decade.

**Note:** This repository is currently in the **Legacy** stage (archived version of the 10th anniversary). Development for the 11th anniversary is ongoing in the `master` branch.

## üöÄ Getting Started

### Prerequisites

* Node.js (v18+ recommended)
* npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone [https://github.com/lingyunmo/lomAnniversary.git](https://github.com/lingyunmo/lomAnniversary.git)
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Development

To start the local development server:

```bash
npm run dev
```

### Production Build

To build the project for production (this will generate the `dist` folder):

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```
